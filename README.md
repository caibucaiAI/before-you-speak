# before-you-speak · 开口之前

你是不是也这样：话在心里过了一百遍，手举到一半又放下。明明想说的都有了，还是觉得自己没准备好，想练到 80 分再开口。于是机会就躲在「再准备一下」里，一次一次，让给了别人。

这个 skill 想把这件事反过来：**60 分敢开口，好过 80 分不敢上台。**

它不是 PPT 生成器，也不帮你把稿子写漂亮。它管的是上台前的最后一公里：稿子捋顺、彩排反馈变成具体改法、设备提前排雷、给你一张攥在手里的候场卡。卡住了有话接，讲不完知道删哪，忘词了低头看一眼。有安全网的人，才敢开口。

> 很多事情，是在你勇敢表达之后才开始的。

## 适合谁

- 要做演讲、分享、路演、圆桌、工作坊，但一想到上台就紧张的人
- 只是要做个自我介绍、说段敬酒词、几分钟小发言，也会提前好几天开始慌的人
- 彩排完收到一堆「有点乱」「太平了」的反馈，不知道先改哪的人
- 明天就要上台、现在只想要一张小抄的人
- 讲完想把经验留下来，下次不用从零开始怕的人

尤其想帮到那些总把自己 push 到 80 分才敢开口的人。

## 怎么用

装好之后随便说一句就行：

- 「我下周要做一场分享，好紧张」
- 「彩排完了，朋友说我讲得乱」（把反馈、截图、稿子直接丢进来）
- 「我明天就要上台，救命」
- 「下个月有个圆桌，我一被 cue 就脸红」
- 「讲完了，帮我复盘」
- 「我又要上台了」（老用户回访，它会带着你上次的经验来）

不用填任何表格。它会先给你一个能用的东西（比如两个开场候选），再问最多两个问题。

## 它会陪你走四步

| 步骤 | 在做什么 |
|---|---|
| 🎬 排练厅 | 灯还没亮，说错不算数。捋主线、改稿子、消化彩排反馈 |
| 🔦 试灯 | 设备替你过一遍：全屏、翻页、投屏、二维码、打包 |
| 🎫 侧幕 | 离光一步。手里一张候场卡：第一句、主线、救场句 |
| 🌙 灯暗之后 | 掌声收好，经验写进返场报告 |

时间紧就直接进侧幕拿候场卡，时间充裕就从排练厅走全程。自我介绍、敬酒词这类轻场景不走四步，直接出卡。

## 它会越用越懂你：场记本

每场讲完，复盘会写进一份**返场报告**（返场的意思是：观众喊了 encore，你还会再上台），报告再沉淀进你的**场记本**：验证过的开场、你自己的金句、实测有效的救场句、下一场要试的一件事。

下次你说「我又要上台了」，它先翻你的场记本，候场卡直接带上你验证过的东西：「你上次那句开场观众笑了，这次还用它打底。」讲得越多，它越像你自己的经纪人，而不是一个模板。

它还内置了一份从《演讲之禅》《演讲的力量》蒸馏的 27 条真实案例库（名人怯场、现场事故、被问倒、开场收尾、练习方法），在你需要的时候按需取用。但你自己验证过的经验，永远排在大师案例前面。

## 首次使用会发生什么

第一次启动时，它会先介绍自己是谁、为什么存在、你可以先发什么。之后再用，它不会重复自我介绍，直接开始干活。

讲完之后的返场报告默认问你想存在哪个文件夹，之后都存那里。
